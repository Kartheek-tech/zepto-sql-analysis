# Zepto Product Analysis (SQL Project)

## 📊 Overview
This project analyzes Zepto product data using SQL to extract business insights.

## 🧾 Dataset
- Product category
- Product Name
- Price (MRP, discounted)
- Stock availability
- Weight

## 🔍 Steps Performed
1. Data Exploration
2. Data Cleaning
3. Data Transformation
4. Business Analysis

## 🔥 Key Insights
- High-value products are frequently out of stock
- Some categories offer higher average discounts
- Inventory weight varies significantly across categories

## 🛠 Tools Used
- MySQL
- SQL

## 📁 Files
- `zepto_analysis.sql` → All queries
- `dataset.csv` → Raw data
